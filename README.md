# 🧭 WcWay — 공중화장실 위치 정보 서비스
> Team Five Guys | Team 3조 : 이현무, 김호현, 배태민, 이명근, 유원영  
> 프로젝트 기간: 2025.09.15 ~ 2025.10.14

---

## 📌 프로젝트 개요

### 🔹 프로젝트 주제 및 선정 배경
도시 내 공중화장실 위치 정보를 손쉽게 확인할 수 있는 서비스의 부재로 인해,  
사용자 편의성과 접근성을 향상시키기 위해 **공중화장실 위치 정보 제공 웹 서비스**를 기획했습니다.

### 🔹 기획 의도
사용자가 **직관적이고 편리한 방식으로 공중화장실 정보를 확인**할 수 있도록  
실시간 위치 기반의 지도 서비스를 제공합니다.

### 🔹 주요 내용
- 전국 공중화장실 데이터를 수집 및 정제하여 실시간으로 제공
- 카카오맵 API를 이용한 위치 시각화
- 로그인 및 회원가입 기능을 통한 사용자 맞춤형 서비스 제공
- 반응형 웹을 통한 접근성 향상
